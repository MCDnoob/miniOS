# qemu+vmware

安装`qemu`作为模拟器，并且用`qemu-gdb`进行调试，这是与`bochs`并行的一种选择。

## 安装qemu
```bash
sudo apt install qemu-system qemu-user-static
```

## Makfile增加对应的选项
```Makefile
.PHONY: qemu
qemu: $(BUILD)/master.img
	qemu-system-i386 \
	-m 32M \
	-boot c \
	-hda $<
```

gdb调试版：
```Makefile
.PHONY: qemu-gdb
qemu-gdb: $(BUILD)/master.img
	qemu-system-i386 \
	-m 32M \
	-boot c \
	-hda $< \
	-s -S
```
