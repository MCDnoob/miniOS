# 主引导扇区

## BIOS

Basic Input Output System

BIOS在加电自检将主引导扇区读`0x7c00`

    int 0x10; BIOS的中断，是显示器相关功能，用于清除显示器内容

x86-开机-BIOS-实模式-内核加载-保护模式
- Read Mode
- Protected Mode

## 实模式的寻址方式

> 有效地址 = 段地址(基址) * 16 + 偏移地址

```s
; 0xb8000屏幕文本显示器的内存
mov ax, 0xb800
mov ds, ax
mov byte [0], 'H'
```

所以：
> EA = 0xb800 * 0x10 + 0 = 0xb8000
EA(Effective Address)

## 主引导扇区的结构
共512B
- 代码: 446B
- 硬盘分区表: 64B = 4 * 16B
- 魔数: 0xaa55
